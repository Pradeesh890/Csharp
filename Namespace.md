Namespace is a collection of classes and "other types such as interfaces, structures, delegate types, enumerations"

We have nested namespace the goal of nested namespace is to create sub group of classes using namespace

By using "Using NamespaceName" we can import the namespace in other programs instead of using .every time

The "Using alias = NamespaceName" directive allows the developers to create alias for long namespace names.

##### "Using static" directive
'""""""""""""""""syntax'"""""""""""""""""

```
using static NamespaceName.StaticClassName
```

the main goal of "using static" is for accessing all the static properties and static methods of static class without mentioning the static class name.
