.Net is a Developer Platform in which developers can build Cross platform projects
.Net includes 
 - Runtime
 - Libraries
 - SDKs
 - Tools

It supports many languages
- C#
- VB.NET
- F#

.Net uses [[CLR]] as its runtime

.Net has another infrastructure which tells us how the program should be executed
which is called as [[CLI]]

In the first step of Execution the Code is converted into [[MSIL or IL]] and proceeds to Converted into Machine Code at runtime by CLR (JIT)

.Net languages -> MSIL/IL -> CLR -> Machine Code

[[DotNet Framework]] is the original first developed version of .Net

Then again a subset of .Net is created which is called as [[DotNet Core]]

This primary language use in .Net is [[C Sharp]]
