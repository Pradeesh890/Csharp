Variable that are declared in the class is called as field
syntax of the fields

"access modifier"  "modifier"  " type"  "field name"

access modifiers of field are
1. private
2. protected
3. private protected
4. internal
5. protected internal
6. public

modifiers of the field are
1. static fields are common to all object
2. const fields value cannot be modified -> compiler replaced all constant names with respective value
3. readonly field value cannot be modified -> compilation time restriction only

type name 
1. any primitive types or non primitive type
