The "System.Object" is a pre defined class, which is the "ultimate super class (base class)" in .Net.

for all the structures and enumerations the default parent class is "class System.ValueType" and for "class System.ValueType" "class System.Object" is the parent.

for all the classes and interfaces "class System.Object" is the default parent.

Derived classes can override equals, gethashcodes and tostring methods of object class.

system.object class is meant for achieving "type safety in C#".
