CLI stands for "Common Language Infrastructure"
it is an ECMA/ISO Standard
this defines how .Net languages should work together

This specifies 
CLR
CTS (Common Type system)
CLS (Common Language Specification)
MISL

The components of CLI are 
- [[CTS]] (Common Types System)
- [[CLS]] (Common Language Specification)
- [[MSIL or IL]] (Microsoft Intermediate Language)
- Metadata
- Virtual Execution System
